load(data: dict, validate: bool = False) → Scene¶

Load the data into a raillabel.Scene and return it.

Parameters:

  • data (dict) – A dictionary loaded from a JSON-file.

  • validate (bool, optional) – If True, the annotation data is validated via the respective schema. This is recommended, if you are working with a modified or non-official file. Setting this option will increase loading time. Default is False.

Returns:

scene – The loaded scene with the data.

Return type:

raillabel.Scene

Raises:

raillabel.exceptions.SchemaError – if validate is True and the data does not validate against the schema.
